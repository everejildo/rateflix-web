# RateFlix
Plataforma web para **rankear, calificar y reseñar películas**, desarrollada con **Laravel**.  
El proyecto permite a los usuarios explorar películas, consultar rankings, guardar favoritos y publicar reseñas, integrando datos desde una API externa.
Repositorio pensado como proyecto de **portafolio profesional** enfocado en desarrollo web full stack.

## Descripción del proyecto
**RateFlix** es una aplicación web inspirada en plataformas de reseñas cinematográficas.  
Permite a los usuarios interactuar con un catálogo de películas, visualizar rankings dinámicos y generar contenido mediante comentarios y calificaciones.
El sistema implementa operaciones **CRUD completas**, manejo de usuarios y persistencia de datos en una base de datos relacional.

## Funcionalidades principales
- Autenticación de usuarios (registro e inicio de sesión).
- Visualización de películas mediante integración con **TMDB API**.
- Sistema de calificación y reseñas por usuario.
- CRUD de comentarios y valoraciones.
- Gestión de películas favoritas.
- Ranking de películas mejor calificadas.
- Navegación por categorías.
- Perfil de usuario con historial de interacción.
- Acceso público a reseñas sin necesidad de iniciar sesión.

## Tecnologías utilizadas
- **Framework:** Laravel  
- **Lenguaje:** PHP  
- **Frontend:** Blade, HTML5, CSS3, JavaScript  
- **Base de datos:** MySQL (phpMyAdmin)  
- **Entorno local:** Laragon  
- **Control de versiones:** Git & GitHub  
- **Editor:** Visual Studio Code  

## Base de datos
- Base de datos relacional en **MySQL**.
- Manejo de relaciones entre usuarios, películas, reseñas y favoritos.
- Migraciones y modelos implementados en Laravel (Eloquent ORM).
